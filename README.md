# Azure Terraform Labs

This repository contains my hands-on Azure and Terraform labs.

I use this repository to practice cloud infrastructure, networking, Infrastructure as Code, and automation workflows.

## Labs

| Lab | Description |
|---|---|
| [lab-01-networking](./lab-01-networking) | Basic Azure networking with Terraform: Resource Group, Virtual Network, subnets, Network Security Group and NSG association |
| [lab-02-linux-vm](./lab-02-linux-vm) | Linux virtual machine with Terraform: VNet, subnet, NSG rule for SSH, Public IP, Network Interface and Ubuntu VM |
| [lab-03-nginx-web-server](./lab-03-nginx-web-server) | Nginx web server on an Azure Linux VM with Terraform |
| [lab-04-container-instance](./lab-04-container-instance) | Azure Container Instance running a containerized Nginx image with Terraform |
| [lab-05-docker-flask](./lab-05-docker-flask) | Custom Dockerfile and Flask app, built and tested locally with Docker |
| [lab-06-bash-scripting](./lab-06-bash-scripting) | Bash script to check Docker container status and port availability |
| [lab-07-cicd-pipeline](./lab-07-cicd-pipeline) | GitHub Actions CI pipeline validating Terraform configs, Docker build and Bash checks on every PR |
